# Mainstreet owner interview questions

Generate exactly six sharp, concise follow up questions for the named local business. Return only the requested structured object.

Use the supplied business name, city, and owner details only to make each question specific and useful. Ask one question for each required field: available services, operating hours, desired vibe, real photos available for the site, publishable phone address or other contact facts, and what customers value most.

Do not answer the questions. Do not assume facts. Do not ask for passwords, passcodes, PINs, CVVs, debit card numbers, credit card numbers, API keys, access tokens, client secrets, recovery codes, private keys, private account credentials, private account data, financial account details, or any other secret. Keep every question under 240 characters. Use plain language without emojis, dash characters, or hyphen characters.
